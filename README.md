# 🏦 Banking Customer Analytics | End-to-End Data Analytics Project

---

## 📌 Project Overview
This project analyzes banking customer data to uncover insights related to **customer behavior, risk exposure, financial performance, and business growth opportunities**.

It is a complete **end-to-end data analytics project**, covering:
- Data Cleaning (Excel & SQL)
- Data Analysis (Python)
- Data Visualization (Power BI)

---

## 🎯 Problem Statement
Banks need to:
- Identify **high-value customers**
- Understand **risk distribution**
- Analyze **loan and deposit behavior**
- Optimize **regional and branch performance**

This project provides **data-driven insights** to support better decision-making.

---

## 🛠️ Tools & Technologies
- 📊 Excel – Initial data cleaning & overview  
- 🗄️ MySQL – Data transformation & querying  
- 🐍 Python (Pandas, Matplotlib, Seaborn) – EDA & feature engineering  
- 📈 Power BI – Dashboard & visualization  

---

## 🔄 Project Workflow

Excel → SQL → Python → Power BI

---

## 📊 Dataset Description
The dataset contains:
- Customer demographics (Age, Gender, Region)
- Financial data (Income, Loans, Deposits, Net Worth)
- Risk indicators (Credit Score, DTI, Risk Category)
- Banking relationship details

## 📈 Exploratory Data Analysis (Python)

### 🔹 Univariate Analysis
- Risk distribution → Majority are Medium Risk  
- Income distribution → Right-skewed  
- Net worth → High variation  

### 🔹 Bivariate Analysis
- Income vs Loan → Positive relationship  
- DTI vs Risk → Higher DTI = Higher risk  
- Credit Score vs Risk → Strong impact  

### 🔹 Correlation Analysis
- Strong positive:
  - Income ↔ Net Worth  
  - Loan Amount ↔ Income  

- Negative:
  - Credit Score ↔ Risk
